jQuery Sheep Factory

wtf? jQuery Sheep Factory is an attempt to be a gif animation builder tool. It uses jQuery, the jsGIF stack and the power of CSS3. It lets you pick and organize images of your choice in a given sheeps set (!), rotate, flip, control the delay, preview the result, and ultimately test it against the jQuery Sheep plugin!

Generated GIFs can eventually be saved on imgur.com images hosting service for later sharing.
